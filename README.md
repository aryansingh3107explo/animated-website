# AI-Powered Interactive Product Landing Page

An animated skincare product website inspired by Hyphen's Golden Hour Serum.

## Features

- Smooth scroll animations
- Interactive product showcase
- Modern responsive UI
- Product benefits section
- Application guide section
- Premium skincare-inspired design
- Mobile-friendly layout

## Tech Stack

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Live Demo

https://aryansingh3107explo.github.io/animated-website/

## Project Overview

This project recreates a premium skincare brand experience using modern web design principles. The website focuses on immersive storytelling, elegant animations, and responsive layouts to showcase a skincare serum product.

## Installation

```bash
git clone https://github.com/aryansingh3107expo/animated-website.git
```

Open `index.html` in your browser.
